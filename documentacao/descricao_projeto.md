# Análise de Desempenho Acadêmico

## Descrição

O projeto consiste em um sistema desenvolvido em Python para análise do desempenho acadêmico de uma turma.

A aplicação permite cadastrar alunos, registrar notas, calcular médias, classificar situações acadêmicas e gerar relatórios automatizados em Excel, além de gráficos para análise visual dos resultados.

## Objetivo

O objetivo do projeto é aplicar conceitos de programação e análise de dados para transformar informações acadêmicas em relatórios organizados e indicadores de desempenho.

## Tecnologias utilizadas

- Python
- OpenPyXL
- Matplotlib
- LibreOffice Calc / Microsoft Excel

## Funcionalidades

- Cadastro de alunos;
- Registro de quatro notas por aluno;
- Cálculo automático da média;
- Classificação da situação acadêmica;
- Geração de estatísticas da turma;
- Criação de ranking dos alunos;
- Exportação de relatório em Excel;
- Geração de gráficos de desempenho.

## Critérios de avaliação

A classificação dos alunos segue as regras:

- Média maior ou igual a 7:
  - Aprovação

- Média entre 5 e 6.9:
  - Recuperação

- Média menor que 5:
  - Reprovação
