# Funcionamento do Sistema

## Fluxo da aplicação

1. O usuário informa a quantidade de alunos que serão cadastrados.

2. Para cada aluno são informados:
   - Nome;
   - Quatro notas.

3. O sistema calcula a média utilizando:

(média) = soma das notas / quantidade de notas

4. Após o cálculo, o aluno recebe uma classificação:

- Aprovação;
- Recuperação;
- Reprovação.

5. O sistema gera:

- Relatório Excel;
- Estatísticas da turma;
- Ranking dos alunos;
- Gráficos de desempenho.

## Arquivos gerados

### Relatórios

Arquivo:

relatorio_desempenho_academico.xlsx

Contém:

- Aba Alunos;
- Aba Estatísticas;
- Aba Ranking.

### Gráficos

São gerados:

- Gráfico de médias dos alunos;
- Gráfico de situação da turma.
